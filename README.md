# Safe-Ed: AI-Powered Educational Ecosystem

Safe-Ed is an all-in-one educational platform designed to bridge the gap between AI-driven content generation and secure classroom management. Built with **Streamlit**, **SQLite**, and **Google Gemini 2.5 Flash**, it provides specialized toolkits for both Teachers and Students.
